# Cave à vin partagée (add-on Home Assistant)

Add-on Home Assistant pour gérer une cave à vin partagée :

- gestion des caves et emplacements
- fiches vin
- OCR d'étiquette
- suivi du stock
- enrichissement IA (optionnel via clé Anthropic)

Configuration add-on :

```yaml
anthropic_api_key: ""
```
